# 📄 Factures Fournisseurs

Gestion automatisée des factures fournisseurs avec extraction IA et synchronisation Supabase.

## 🚀 Fonctionnalités

- **Dashboard** : métriques en temps réel (à payer, retard, alertes J-3)
- **Extraction IA** : dépôt PDF → extraction automatique des données
- **Synchronisation Supabase** : lecture des factures depuis la base de données
- **PWA** : installable sur ordinateur et mobile
- **Recherche & filtres** : par fournisseur, statut, date
- **Historique** : graphique des paiements mensuels

## 🔧 Installation

1. Clone ce repo
2. Ajoute les fichiers sur Vercel
3. Configure les variables d'environnement :
   - `SUPABASE_URL` : https://mimuqswvyhgsasvbyem.supabase.co
   - `SUPABASE_KEY` : ta clé publique Supabase

## 📊 Stack

- Frontend : HTML/CSS/JavaScript pur
- Hébergement : Vercel
- Base de données : Supabase
- Extraction IA : Make + Claude API
- Automatisation : Make
